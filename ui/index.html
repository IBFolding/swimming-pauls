🐟 WHAT IS SWIMMING PAULS?

Swimming Pauls is a multi-agent prediction engine that runs entirely on your machine. No cloud. No API keys. No data leaving your computer.

You ask a question. The Pauls argue about it. They debate, deliberate, and eventually reach consensus. Each Paul has a unique perspective:

  • 👨‍🏫 Professor Paul - Academic analysis
  • 📈 Trader Paul - Market sentiment
  • 🤨 Skeptic Paul - Risk management
  • 🔮 Visionary Paul - Long-term view
  • 🐋 Whale Paul - Institutional thinking
  • 🎰 Degen Paul - YOLO energy
🎯 How It Works
  1. Connect your local agent
  2. Ask any question
  3. Pauls simulate & debate
  4. Get consensus + confidence
🔒 Privacy First
Everything runs locally. Your questions never hit a server. Your data stays on your machine. Even the AI models run locally.

🔌 Connect Local Agent

Run the Swimming Pauls agent on your machine and connect it to this UI.

# 1. Clone and install
git clone github.com/IBFolding/swimming-pauls
cd swimming-pauls
pip install -r requirements.txt
# 2. Start local agent
python local_agent.py
# 3. Connection ID will appear
