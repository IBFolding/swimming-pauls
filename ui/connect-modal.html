🔌 Connect Local Agent

Run the Swimming Pauls agent on your machine:

1. Clone (if needed)
cd ~ && git clone https://github.com/IBFolding/swimming-pauls.git
2. Install (first time)
pip3 install websockets
3. Start Agent
cd ~/swimming-pauls && python3 local_agent.py
4. Connect
💡 Tips:
  • Use Demo Mode to test without Python
  • Connection ID appears in terminal when agent starts
  • If python3 doesn't work, try python
